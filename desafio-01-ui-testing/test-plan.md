# Plano de Testes - Sauce Demo

## 📌 Objetivo

Validar as funcionalidades principais da aplicação Sauce Demo, garantindo o correto funcionamento dos fluxos de autenticação, navegação, gerenciamento de produtos, carrinho e checkout.

---

# 🎯 Escopo dos Testes

## Funcionalidades cobertas

- Login
- Logout
- Navegação entre páginas
- Gerenciamento de produtos
- Ordenação de produtos
- Carrinho de compras
- Processo de checkout

---

# 🧪 Tipos de Teste

- Teste funcional
- Teste exploratório
- Teste negativo
- Teste de usabilidade
