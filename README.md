# QA Testing Challenge - BetaTalent

Este repositório contém a resolução completa do teste prático de QA Testing proposto pela BetaTalent, dividido em duas etapas principais:

- Desafio 01 - UI Testing (SauceDemo)
- Desafio 02 - API Testing (Restful Booker)

O objetivo do projeto foi validar funcionalidades críticas das aplicações, documentar cenários de teste, identificar riscos e bugs, além de implementar automações e evidências dos testes executados.
